# مسار التخفي (Mindustry Mod)

يحسب ويرسم أكثر المسارات أماناً / أقل ضرراً باستخدام مدى أبراج ووحدات العدو.

الإصدار الحالي: `2.0.6`

## الاستخدام

### مفاتيح الاختصار

- `X` (`sp_path_turrets`): الأبراج فقط (اضغط مطولاً = معاينة مباشرة)
- `Y` (`sp_path_all`): الأبراج + الوحدات (اضغط مطولاً = معاينة مباشرة)
- `N` (`sp_auto_mouse`): وضع تلقائي (مجموعة → الماوس)
- `M` (`sp_auto_attack`): وضع تلقائي (مجموعة → <Attack>(x,y))
- `K` (`sp_mode_cycle`): اضغط للتبديل بين أوضاع العرض
- `L` (`sp_threat_cycle`): تبديل مُرشّح التهديد (أرض/جو/كلاهما)

### الأوضاع التلقائية

- يستخدم مركز مجموعة الوحدات (الوحدات المحددة إن وجدت؛ وإلا وحدتك الحالية) ويكشف تلقائياً جو/أرض/مختلط لحساب مسار أقل ضرراً (وضع Y: أبراج + وحدات).
- إذا كانت الوحدات المحددة مقسمة إلى عدة مجموعات تفصل بينها مسافة أكبر من العتبة المُعدّة (الافتراضي 5 مربعات)، فسيحسب ويرسم مساراً لكل مجموعة على حدة.
- أثناء تنفيذ أمر الحركة التلقائية، يقوم المود بتحديث نقاط RTS عندما يتغير أفضل مسار (مثلاً عندما يبدأ برج بإطلاق النار).
- هدف عبر الدردشة: أرسل `"<Attack>(x,y)"` في الدردشة (x,y إحداثيات البلاطة). يتم تحليل رسالتك أنت فقط (على العميل).

### الأوضاع

اضغط `K` للتبديل بين 3 أوضاع:

1) وضع الهدف العادي (يُختار من الإعدادات)
2) مجموعات مولدات طاقة العدو (يرسم عدة مسارات)
3) اللاعب -> موقع المؤشر

## الإعدادات

افتح: `Settings → Stealth Path`

- مدة عرض المسار
- سماكة الخط
- معدل تحديث المعاينة (عند الضغط المطوّل على X/Y)
- وضع احترافي: يفتح قائمة الإعدادات المتقدمة
- الإعدادات المتقدمة (الوضع الاحترافي): ضبط معاملات البحث عن المسار/الحركة التلقائية/الذاكرة المؤقتة/مجموعات المولدات
- وضع المولدات: اللون، عدد المسارات، أقل حجم للمجموعة، البدء من اللاعب
- وضع اللاعب->المؤشر: اللون

## اللعب الجماعي

هذا مود واجهة (Overlay) على جهة العميل فقط. يستخدم `mod.json` القيمة `"hidden": true` لتجنب فحص اختلاف المودات بين الخادم والعميل.

## البناء

من مجلد جذر مستودع `Mindustry-master`:

```powershell
./gradlew.bat stealth-path:jar
```

الملف الناتج: `mods/stealth-path/build/libs/stealth-path.zip`
