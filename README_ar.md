# مسار التخفي (Mindustry Mod)

يحسب ويرسم أكثر المسارات أماناً / أقل ضرراً باستخدام مدى أبراج ووحدات العدو.

## الاستخدام

### مفاتيح الاختصار

- `X` (`sp_path_turrets`): الأبراج فقط (اضغط مطولاً = معاينة مباشرة)
- `Y` (`sp_path_all`): الأبراج + الوحدات (اضغط مطولاً = معاينة مباشرة)
- `K` (`sp_mode_cycle`): اضغط للتبديل بين أوضاع العرض
- `L` (`sp_threat_cycle`): تبديل مُرشّح التهديد (أرض/جو/كلاهما)

### الأوضاع

اضغط `K` للتبديل بين 3 أوضاع:

1) وضع الهدف العادي (يُختار من الإعدادات)
2) مجموعات مولدات طاقة العدو (يرسم عدة مسارات)
3) اللاعب -> موقع المؤشر

## الإعدادات

افتح: `Settings → Stealth Path`

- مدة عرض المسار
- سماكة الخط
- معدل تحديث المعاينة (عند الضغط المطوّل على X/Y)
- وضع المولدات: اللون، عدد المسارات، أقل حجم للمجموعة، البدء من اللاعب
- وضع اللاعب->المؤشر: اللون

## اللعب الجماعي

هذا مود واجهة (Overlay) على جهة العميل فقط. يستخدم `mod.json` القيمة `"hidden": true` لتجنب فحص اختلاف المودات بين الخادم والعميل.

## البناء

من مجلد جذر مستودع `Mindustry-master`:

```powershell
./gradlew.bat stealth-path:jar
```

الملف الناتج: `mods/stealth-path/build/libs/stealth-path.zip`
