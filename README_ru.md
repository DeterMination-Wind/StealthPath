# Скрытный путь (Mindustry Mod)

Вычисляет и рисует самые безопасные / наименее опасные маршруты, используя радиус действия вражеских турелей и юнитов.

Текущая версия: `2.0.0`

## Использование

### Горячие клавиши

- `X` (`sp_path_turrets`): только турели (удерживать = предпросмотр)
- `Y` (`sp_path_all`): турели + юниты (удерживать = предпросмотр)
- `N` (`sp_auto_mouse`): авто-режим (отряд → курсор)
- `M` (`sp_auto_attack`): авто-режим (отряд → <Attack>(x,y))
- `K` (`sp_mode_cycle`): нажмите, чтобы переключить режим отображения
- `L` (`sp_threat_cycle`): переключить фильтр угроз (земля/воздух/оба)

### Авто-режимы

- Использует центр отряда (выделенные юниты, если есть; иначе ваш текущий юнит) и автоматически определяет воздух/земля/смешанный, чтобы вычислить путь с минимальным уроном (режим Y: турели + юниты).
- Цель через чат: отправьте `"<Attack>(x,y)"` в чат (x,y — координаты клетки). Разбирается только ваше собственное сообщение (клиент).

### Режимы

Нажимайте `K`, чтобы переключать 3 режима:

1) Обычный режим цели (выбирается в настройках)
2) Группы вражеских генераторов (рисует несколько путей)
3) Игрок -> позиция курсора

## Настройки

Открыть: `Settings → Stealth Path`

- Время показа пути
- Толщина линии
- Интервал обновления предпросмотра (при удержании X/Y)
- Режим генераторов: цвет, количество путей, минимальный размер группы, старт от игрока
- Режим игрок->курсор: цвет

## Мультиплеер

Это клиентский overlay-мод. В `mod.json` используется `"hidden": true`, чтобы избежать проверок несовпадения модов сервер/клиент.

## Сборка

Из корня репозитория `Mindustry-master`:

```powershell
./gradlew.bat stealth-path:jar
```

Выход: `mods/stealth-path/build/libs/stealth-path.zip`
